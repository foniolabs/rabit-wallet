## What you get

<FeatureGrid>
  <Feature icon="🔐" title="Embedded auth">
    Email, social, passkey, and external wallet — all in one drop-in component. No redirect loops.
  </Feature>
  <Feature icon="✨" title="Smart accounts, built in">
    ERC-4337 by default. Sponsor gas, batch transactions, recover keys.
  </Feature>
  <Feature icon="🌍" title="Multi-chain">
    EVM and Solana, one consistent API. Switch networks, sign, send.
  </Feature>
  <Feature icon="🔑" title="Non-custodial keys">
    TSS / MPC. Users own their keys. You never see them.
  </Feature>
</FeatureGrid>

## Install

<InstallTabs pkg="@rabit/react @rabit/core" />

Then wrap your app and drop in a `<ConnectButton />`:

```tsx filename="app/layout.tsx"
import { RabitProvider, ConnectButton } from '@rabit/react'

export default function App() {
  return (
    <RabitProvider apiKey={process.env.NEXT_PUBLIC_RABIT_KEY!}>
      <ConnectButton />
    </RabitProvider>
  )
}
```

<Note tone="tip" title="Already have a project?">
  You can install `@rabit/react` into any React app — Next.js, Vite, Remix, CRA. The API is identical.
</Note>
