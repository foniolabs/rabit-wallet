# Theming

The `<RabitProvider>` accepts a `theme` prop that controls the connect modal, button, and account widgets.

```tsx
<RabitProvider
  apiKey={key}
  theme={{
    mode: 'dark',
    accent: '#FF7A1A',
    radius: 'lg',
    font: 'Inter, sans-serif',
  }}
>
  {children}
</RabitProvider>
```

## Tokens

| Token   | Type                              | Default          |
| ------- | --------------------------------- | ---------------- |
| `mode`  | `'light' \| 'dark' \| 'auto'`     | `'auto'`         |
| `accent`| CSS color                         | `#5B5BD6`        |
| `radius`| `'none' \| 'sm' \| 'md' \| 'lg'`  | `'md'`           |
| `font`  | CSS font stack                    | system stack     |

## Full custom UI

If the default UI doesn't fit, drop down to hooks and build your own:

```tsx
import { useConnect, useAccount } from '@rabit/react'

function MyConnect() {
  const { connectors, connect, isPending } = useConnect()
  const { isConnected } = useAccount()
  if (isConnected) return null
  return connectors.map((c) => (
    <button key={c.id} onClick={() => connect({ connector: c })} disabled={isPending}>
      Connect {c.name}
    </button>
  ))
}
```
