# Core API

`@rabit/core` is the framework-agnostic SDK. Use it directly in vanilla JS, Vue, Svelte, or any non-React environment.

```bash
npm i @rabit/core
```

```ts
import { createRabit } from '@rabit/core'

const rabit = createRabit({
  apiKey: process.env.RABIT_KEY!,
  chains: ['ethereum'],
})

await rabit.connect()
const { address } = rabit.getAccount()
const hash = await rabit.sendTransaction({ to: '0x…', value: 1n })
```

## Surface

| Method                | Description                                  |
| --------------------- | -------------------------------------------- |
| `createRabit(config)` | Construct a Rabit instance.                  |
| `.connect()`          | Open the auth flow, return the account.      |
| `.disconnect()`       | Sign out the current account.                |
| `.getAccount()`       | Read current account state synchronously.    |
| `.subscribe(fn)`      | Subscribe to account / chain changes.        |
| `.sendTransaction()`  | Send an EVM transaction.                     |
| `.signMessage()`      | Sign an arbitrary message.                   |
