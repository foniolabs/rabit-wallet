# Hooks

| Hook                  | Returns                                                                          |
| --------------------- | -------------------------------------------------------------------------------- |
| `useAccount()`        | `{ address, isConnected, chainId, status }`                                      |
| `useConnect()`        | `{ connectors, connect, isPending, error }`                                      |
| `useDisconnect()`     | `{ disconnect, isPending }`                                                      |
| `useBalance(args?)`   | `{ data, isLoading, refetch }`                                                   |
| `useSendTransaction()`| `{ sendTransaction, isPending, data, error }`                                    |
| `useSignMessage()`    | `{ signMessage, signMessageAsync, isPending, data, error }`                      |
| `useSwitchChain()`    | `{ switchChain, chains, isPending }`                                             |
| `useSendCalls()`      | `{ sendCalls, isPending, data }` — batched smart-account calls                   |
