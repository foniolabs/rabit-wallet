# React API

`@rabit/react` is the recommended entry point for React apps. It re-exports the framework-agnostic `@rabit/core` and adds:

- A provider — [`RabitProvider`](/react/provider)
- [Hooks](/react/hooks) — `useAccount`, `useConnect`, `useSendTransaction`, …
- [Components](/react/components) — `ConnectButton`, `AccountModal`, …

```bash
npm i @rabit/react
```
