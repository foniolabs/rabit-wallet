<Note tone="info">
  Rabit works with **any React app** — Next.js, Vite, Remix, CRA. Examples below use Next.js, but the API is identical everywhere.
</Note>

<Steps>

### Install

<InstallTabs pkg="@rabit/react @rabit/core" />

### Get an API key

Sign in at [dashboard.rabit.dev](https://dashboard.rabit.dev) and create a project. Copy your **Publishable Key** — it's safe to expose on the client.

```bash filename=".env.local"
NEXT_PUBLIC_RABIT_KEY=pk_live_xxxxxxxxxxxx
```

### Wrap your app

```tsx filename="app/layout.tsx"
import { RabitProvider } from '@rabit/react'

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <RabitProvider apiKey={process.env.NEXT_PUBLIC_RABIT_KEY!}>
          {children}
        </RabitProvider>
      </body>
    </html>
  )
}
```

### Add a Connect button

```tsx filename="app/page.tsx"
'use client'
import { ConnectButton, useAccount } from '@rabit/react'

export default function Home() {
  const { address, isConnected } = useAccount()

  return (
    <main>
      <ConnectButton />
      {isConnected && <p>Hello, {address}</p>}
    </main>
  )
}
```

### Run it

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and click **Connect Wallet**. You're done.
