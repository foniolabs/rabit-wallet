# @rabit/types

TypeScript types for the [Rabit embedded wallet SDK](https://github.com/web3normad/rabit).

```bash
npm i @rabit/types
```

This package only contains type definitions (no runtime code). Most users want **[`@rabit/react`](https://www.npmjs.com/package/@rabit/react)** instead, which re-exports everything you'll typically need.
