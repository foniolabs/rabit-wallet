# @rabit/onramp

On-ramp / off-ramp engine for the [Rabit embedded wallet SDK](https://github.com/web3normad/rabit).

```bash
npm i @rabit/onramp
```

Talks to the Rabit API for live CoinGecko-rate quotes and order management. Used internally by `@rabit/react`'s `useOnRamp()` hook and the `<RabitDashboard />` Buy/Sell pages — most apps don't need to import this directly.
