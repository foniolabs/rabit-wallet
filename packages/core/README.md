# @rabit/core

Core SDK orchestration for the [Rabit embedded wallet](https://github.com/foniolabs/rabit-wallet) — wires together auth, wallet state, smart-account resolution, and on-ramp.

```bash
npm i @rabit/core
```

Exports `RabitCore` — a framework-agnostic class you can drive from any UI layer. React users want **[`@rabit/react`](https://www.npmjs.com/package/@rabit/react)** instead.
